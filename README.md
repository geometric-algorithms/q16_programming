# Q1: Geometric Median 

## How to Run

You can run the program in one of three ways:

### 1. Using a Custom Input File
--> bash test.sh input.txt

input format:
n         : Number of points

x1 y1     : Point 1

x2 y2     : Point 2

...       

xn yn     : Point n


### 2. Provide a number for generating n points randomly
----> bash test.sh 400

### 3. Default case
----> bash test.sh
